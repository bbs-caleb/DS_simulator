## Точный контракт методов

```text
adjust_global_bank_rate → @staticmethod
calculate_avg_balance   → @classmethod
calculate_interest      → @classmethod
```

Проверяющая система проверяет не только результаты, но и конкретный вид каждого метода.

## Что делает класс

Класс `BankMetrics`:

- хранит общую банковскую ставку;
- хранит общий список счетов;
- сохраняет имя и баланс каждого счёта;
- меняет общую ставку;
- считает средний баланс;
- считает проценты для переданного счёта.

## Контрольный пример

```text
Tom:   15000
Jerry: 20000
Spike: 10000
```

Средний баланс:

```text
(15000 + 20000 + 10000) / 3 = 15000
```

После изменения ставки:

```text
global_bank_rate = 16.0
```

Проценты Tom:

```text
15000 × 16 / 100 = 2400.0
```

## Главная идея

- Конкретный объект нужен — обычный метод с `self`.
- Нужен класс — `@classmethod` с `cls`.
- Не нужны автоматически объект и класс — `@staticmethod`.
- В автопроверяемой задаче нужно строго соблюдать ожидаемый API.
