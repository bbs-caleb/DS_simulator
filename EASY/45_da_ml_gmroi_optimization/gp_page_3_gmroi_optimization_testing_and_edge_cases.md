# 5. Почему не добавлена лишняя обработка

Условие не задаёт отдельного поведения для:

- нулевого знаменателя;
- пропущенной категории в pivot;
- отрицательных значений.

Добавление собственных правил может изменить эталон. Решение точно следует формуле и API задания.
