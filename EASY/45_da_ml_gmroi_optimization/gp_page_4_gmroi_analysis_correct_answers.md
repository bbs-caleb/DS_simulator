# Page 4 — Анализ GMROI: правильные ответы

## Какие пункты нужно отметить

Отметьте утверждения:

```text
1, 2, 3, 5, 7, 10
```

То есть правильными являются:

- [x] **Худшие показатели GMROI у продавца 2**
- [x] **На конец года значения GMROI выше начальных у всех продавцов**
- [x] **Худшими по GMROI категориями товаров оказались 2, 4 и 7**
- [x] **Лучшие показатели GMROI в течение года у продавца KarpovExpress**
- [x] **Лучшие категории по GMROI: 1, 5, 6, 8, 9, 10**
- [x] **GMROI всех категорий товаров улучшился на конец года**

Не отмечайте:

- [ ] GMROI всех продавцов выше 1.5 на конец 2023 года
- [ ] Все категории товаров продемонстрировали близкие значения GMROI
- [ ] Сложно выделить наиболее приоритетных продавцов для продвижения
- [ ] Среди закупаемых KarpovExpress категорий сложно выделить аутсайдеров

# Важная оговорка

Утверждения 2 и 10 говорят именно о сравнении:

```text
первая неделя ↔ последняя неделя
```

Они не означают, что GMROI рос непрерывно каждую неделю. В течение года были локальные колебания.
