# 3. Что такое MMM в контексте задачи

Marketing Mix Modeling оценивает вклад управляемых факторов в бизнес-результат.

Классический MMM может анализировать:

- телевизионную рекламу;
- digital-рекламу;
- скидки;
- промо;
- цены;
- сезонность;
- внешние факторы.

В этой учебной задаче логика переносится на закупки:

- признаки — расходы на закупки категорий;
- целевая переменная — GMROI;
- коэффициенты — направление и сила линейной связи.

Это упрощённый учебный MMM. Реальные модели часто учитывают:

- лаги;
- насыщение;
- сезонность;
- тренд;
- промо;
- цены;
- внешние факторы;
- взаимное влияние категорий.

# 4. Почему класс наследуется от GMROI

```python
class CategoryImpactEval(GMROI):
```

Это означает, что `CategoryImpactEval` получает методы и атрибуты класса `GMROI`.

После вызова:

```python
super().__init__(...)
```

внутри объекта уже существуют:

```python
self.seller_gmroi
self.category_gmroi
self.purchases
```

Повторно писать расчёт GMROI и закупок не нужно.

# 8. Зачем нужен StandardScaler

Закупки измеряются миллионами рублей.

Пример:

```text
5 129 792
9 231 286
4 241 830
```

После стандартизации каждая категория переводится примерно в шкалу:

- среднее = 0;
- стандартное отклонение = 1.

Формула:

\[
z =
\frac{x - \mu}{\sigma}
\]

Это делает коэффициенты разных категорий сопоставимыми по масштабу.

---

# 9. Интерпретация коэффициентов

Результат:

```python
{
    1: -0.0020,
    3: -0.0017,
    5: 0.0038,
    8: -0.0001,
    9: -0.0051,
}
```

Поскольку признаки стандартизированы, коэффициент показывает изменение GMROI при росте закупок категории примерно на одно стандартное отклонение при неизменности остальных признаков.

Категория `5` имеет наибольший положительный коэффициент:

```text
+0.0038
```

Категория `9` имеет наибольший по модулю отрицательный коэффициент:

```text
-0.0051
```

Но коэффициенты очень близки к нулю, а R² низкий. Поэтому связи нельзя считать сильными.

---

# 10. Что означает intercept

```text
intercept = 1.5267
```

После стандартизации нулевое значение каждого признака означает средний уровень закупок этой категории.

Поэтому intercept — прогнозируемый GMROI при средних закупках всех категорий.

---

# 11. Что означает R² = 0.0402

\[
R^2 = 0.0402
\]

Модель объясняет примерно:

```text
4.02%
```

вариации недельного GMROI.

Оставшиеся примерно 95.98% колебаний линейная модель закупок не объясняет.

Возможные причины:

- связь нелинейная;
- эффект проявляется с задержкой;
- GMROI зависит от продаж, цен и себестоимости;
- закупки влияют на будущие недели;
- есть сезонность;
- мало наблюдений;
- категории взаимосвязаны;
- линейная модель слишком простая.

---

# 12. Бизнесовый вывод

Нельзя принимать решение:

> Увеличить закупки категории 5 и сократить категорию 9 только из-за коэффициентов.

Причины:

- R² очень низкий;
- коэффициенты малы;
- модель показывает корреляцию, а не причинность;
- закупки и GMROI измеряются в одной неделе без лагов;
- не учтены другие факторы.

Правильный вывод:

> Линейная модель не смогла качественно описать GMROI только через недельные закупки категорий. Следует попробовать более гибкую модель и расширить набор факторов.
