## Главная мысль

GMROI отвечает не на вопрос:

> «Сколько денег мы заработали?»

А на вопрос:

> «Насколько эффективно каждый рубль, вложенный в товарный запас, создаёт валовую прибыль?»

Высокая абсолютная прибыль сама по себе ещё не означает, что товар, категория или продавец эффективно используют капитал.

## Формула

\[
GMROI =
\frac{Gross\ Profit}
{Average\ Inventory\ Cost}
\]

Где:

- `Gross Profit` — валовая прибыль;
- `Average Inventory Cost` — средняя стоимость товарных запасов за период.

Базовый вариант:

\[
Gross\ Profit = Revenue - Cost\ of\ Goods\ Sold
\]

\[
Average\ Inventory\ Cost =
\frac{Beginning\ Inventory + Ending\ Inventory}{2}
\]

## Как читать значение

- `GMROI = 0.8` — каждый рубль среднего запаса создал 0.8 рубля валовой прибыли.
- `GMROI = 1.0` — каждый рубль среднего запаса создал 1 рубль валовой прибыли.
- `GMROI = 2.5` — каждый рубль среднего запаса создал 2.5 рубля валовой прибыли.

Важно: GMROI — не выручка на рубль запасов и не чистая рентабельность бизнеса. Это отношение валовой прибыли к средней стоимости запасов за конкретный период.

## Как изучать материал

1. Прочитать файл `context`.
2. Разобрать формулу и примеры из файла `theory`.
3. Попробовать своими словами объяснить:
   - зачем бизнесу GMROI;
   - почему нельзя ранжировать продавцов только по GMROI;
   - почему нельзя усреднять GMROI обычным средним;
   - чем отличается высокая прибыль от высокой эффективности запасов.
4. Пройти файл по интервью и ответить на вопросы без подсказки.
