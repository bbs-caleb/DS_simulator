# 8. Почему не используется StandardScaler

Для деревьев масштаб признаков не критичен.

Добавление scaler:

- не требуется условием;
- создаёт лишний код;
- может изменить внутренние пороги;
- усложняет объяснение.

---

# 9. Почему прогноз строится на features

Условие ожидает training R²:

```python
predictions = model.predict(features)
```

Если разделить train/test, результат не совпадёт с эталоном.

---

# 10. Почему `verbose=False` допустим

Он влияет только на печать процесса обучения.

Математические параметры модели не меняются.
