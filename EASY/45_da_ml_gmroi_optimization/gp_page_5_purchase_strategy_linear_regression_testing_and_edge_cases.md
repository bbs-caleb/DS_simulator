# 7. Почему нет train/test split

В условии ожидается результат, рассчитанный на всей выборке:

```text
R² = 0.0402
```

Если разделить данные:

- R² изменится;
- коэффициенты изменятся;
- результат не совпадёт с эталоном.

Для реального проекта временное разделение обязательно, но учебный API его не требует.

---

# 8. Почему дата не используется как признак

`week` нужна для merge, но не является закупочной категорией.

Добавление даты:

- изменит модель;
- позволит ловить временной тренд;
- изменит коэффициенты;
- нарушит ожидаемый результат.

---

# 9. Почему scaler обучается на всей выборке

Это соответствует условию и эталонному выводу.

В production scaler нужно обучать только на train-периоде, чтобы избежать утечки.
