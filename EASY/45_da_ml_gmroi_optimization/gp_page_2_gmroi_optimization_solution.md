# Логика решения

## `_load_csv`

- читает CSV через `pd.read_csv`;
- сохраняет `FileNotFoundError`;
- остальные ошибки превращает в `RuntimeError`.

## `_process_gmroi`

1. Проверяет допустимость `group_columns`.
2. Объединяет товары с продажами.
3. Добавляет остатки.
4. Рассчитывает:
   - `margin`;
   - `avg_inventory_cost`;
   - `storage_cost`.
5. Суммирует показатели по выбранной группировке.
6. Рассчитывает GMROI.
7. Округляет до двух знаков.
8. Оставляет только требуемые колонки.
9. Сортирует результат.

## `_process_purchases`

1. Оставляет товары продавца `0`.
2. Присоединяет к закупкам категорию и себестоимость.
3. Рассчитывает стоимость закупки.
4. Суммирует по неделе и категории.
5. Разворачивает категории в столбцы через `pivot`.
6. Сортирует по неделе.

# Почему код минимальный

Использованы только базовые операции pandas:

- `read_csv`;
- `merge`;
- `groupby`;
- `agg`;
- `pivot`;
- `sort_values`;
- `reset_index`.

Нет:

- ручных циклов;
- лишних классов;
- ненужных библиотек;
- изменения API;
- вымышленных правил обработки данных.
