## Страница 1: t-тест для средней ошибки как защитной метрики

Этот комплект относится только к первой странице урока.

На странице вводится большая тема сравнения хвостов распределения ETA-ошибок, но конкретная программная задача пока проверяет более узкое утверждение:

> отличается ли средняя ошибка старой ETA-модели от средней ошибки новой ETA-модели?

Поэтому здесь нужен обычный двухвыборочный t-тест для двух независимых выборок.

## Главное различие

- **Главная бизнес-гипотеза урока:** новая модель уменьшает тяжёлые недопрогнозы в правом хвосте.
- **Задача этой страницы:** проверить, не возник ли неожиданный сдвиг средней ошибки.
- **Роль среднего:** guardrail, то есть защитная метрика.
- **Роль хвостовой метрики:** primary metric эксперимента, но её тестирование будет отдельной задачей.

## Финальная функция

```python
from typing import List, Tuple

from scipy.stats import ttest_ind


def ttest(
    control: List[float],
    experiment: List[float],
    alpha: float = 0.05,
) -> Tuple[float, bool]:
    """Two-sample t-test for the means of two independent samples."""
    _, p_value = ttest_ind(control, experiment)
    result = bool(p_value < alpha)

    return float(p_value), result
```

## Логика в пяти шагах

1. Получаем ошибки старой модели: `control`.
2. Получаем ошибки новой модели: `experiment`.
3. Вызываем `scipy.stats.ttest_ind` с дефолтными параметрами.
4. Сравниваем `p_value` с `alpha`.
5. Возвращаем `(p_value, result)`.

Где:

- `result == True` — различие средних статистически значимо;
- `result == False` — оснований объявить различие средних значимым недостаточно.
