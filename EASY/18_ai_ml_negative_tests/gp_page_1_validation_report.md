# Отчёт локальной проверки решения

Для проверки был создан временный `metrics.py` с функцией из условия и выполнена команда:

```bash
python -m pytest -q
```

Результат:

```text
...... [100%]
6 passed
```

Код завершения `pytest`: `0`.

Это подтверждает, что все шесть тестов проходят на функции `ctr`, приведённой в задании.

> Техническое сообщение среды о прогреве spreadsheet runtime не относится к тестам и не влияет на результат `pytest`.
