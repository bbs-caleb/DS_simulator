## Как проходить материал

Рекомендуемый порядок:

```text
1. gp_page_1_memoization_context.md
2. gp_page_1_memoization_interview.md
3. gp_page_1_memoize_solution.py
4. gp_page_1_memoize_code_explanation.md
```

Не начинайте с механического запоминания кода.

Сначала необходимо понять четыре идеи:

1. Повторное вычисление может стоить дорого.
2. Аргументы функции можно превратить в ключ словаря.
3. Результат можно сохранить как значение словаря.
4. При повторном вызове можно вернуть готовое значение.
