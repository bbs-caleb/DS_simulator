# 17 урок ML: HELLO BERT — теоретический разбор

## Что это

Исходный материал урока — это чистая теория без практического задания и без грейдера
(нет ни одной задачи с проверкой, только концептуальное объяснение BERT). Поэтому вместо
"кода на 100 баллов" здесь — предельно подробный, разложенный по полочкам разбор самой
теории, чтобы дальше свободно ориентироваться в терминах и концепциях, которые понадобятся
на практике (fine-tuning, `[CLS]`-эмбеддинг, MLM, NSP и т.д.).

## Файлы

- `ml_bert_theory_breakdown_01.md` — основной файл разбора. Структура:
  1. Бизнес-контекст (зачем FlyingFood вообще нужен Sentiment Analysis)
  2. Технический контекст (что такое трансформер, эмбеддинг, почему "трансформер" так называется)
  3. Как предобучают BERT (MLM + NSP, с примерами)
  4. Fine-tuning под конкретную задачу (варианты, какой используем в MVP)
  5. Итоговая сборка бизнес- и технической части
  6. Глоссарий терминов для быстрой сверки

## Как читать

Файл рассчитан на чтение с нуля — без предположения, что ты уже знаком с трансформерами
или NLP. Термины вводятся последовательно, каждое новое понятие опирается только на уже
объяснённые ранее.

## Что дальше

Как только у тебя появится следующий шаг урока с конкретным практическим заданием (например,
код на извлечение `[CLS]`-эмбеддингов через `transformers` и обучение классификатора поверх
них, с грейдером) — пришли его отдельным файлом, и я сделаю разбор в стандартном для тебя
формате: бизнес-контекст → технический контекст → готовый рабочий `.py`-файл → пошаговое
объяснение кода с нуля, всё в MD.
