# 📊 ML Lesson 19 — Error Analysis

## О чём этот модуль

Это разбор урока **"Error Analysis"** — построение библиотеки для автоматического и унифицированного анализа ошибок ML-моделей. Кейс — сервис доставки еды **Flying Food**, но подход универсален для любой компании, которая строит прогнозные модели (fintech, e-commerce, маркетплейсы, igaming, bigtech и т.д.).

## Структура файлов

Каждый шаг курса оформляется отдельным `.md` файлом с префиксом `cl_step_N_`:

| Файл | Содержание |
|---|---|
| `README.md` | Этот файл — навигация по модулю |
| `cl_step_1_theory.md` | **Максимально подробная теория**: что такое Error Analysis, зачем он нужен, чем отличается от обычной валидации, остатки и псевдоостатки, план всего модуля, бизнес-применение в разных индустриях |

По мере появления новых заданий (с конкретным кодом) будут добавляться:
- `cl_step_N_*.py` — готовый код к каждому заданию
- `cl_step_N_*.md` — пошаговое объяснение этого кода "с нуля"

## Как читать материалы

1. Файлы читаются **строго по порядку** — каждый шаг опирается на предыдущий.
2. Если в шаге только теория (без задания) — файл содержит расширенный разбор темы с бизнес-контекстом.
3. Если в шаге есть задание — вы получаете: бизнес- и техническую постановку задачи → готовый `.py` файл → построчное объяснение кода.
