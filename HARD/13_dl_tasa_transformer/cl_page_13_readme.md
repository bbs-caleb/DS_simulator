# Урок 13 — DL: TASA Transformer (Real-Time RecSys)

## Что за задача

Вы работаете в команде рекомендаций KarpovExpress. Нужна система, которая адаптирует рекомендации **в реальном времени** — пользователь переключился с пылесосов на колонки, и лента мгновенно перестраивается. За основу берётся архитектура **TASA** (автоэнкодер на последовательностях действий), но LSTM заменяется на **трансформеры** (BERT-энкодер + трансформер-декодер).

## TL;DR решения

```
BERT-энкодер (предобученный) → кодирует src и tgt последовательности
nn.TransformerDecoderLayer + nn.TransformerDecoder → декодирует tgt с оглядкой на src
nn.Linear(hidden_dim, vocab_size) → логиты по всем товарам
```
